db.resource_repo module¶

class src.db.resource_repo.ResourceRepo(con)¶

Bases: object

Methods to manipulate “resource” table in the Goalz database

The sqlite3 connection instance is received as a constructor parameter and is accessible to all the methods of this class through the self.con attribute.

Methods of this class MUST not be accessed directly. All the calls to the database should be made through the API provided by Connection

Parameters:con (sqlite3.Connection) – Connection to an SqlLite database
create_resource(goal_id, user_id, title, link, topic, description, required_time)¶

Create a new resource with the data provided as arguments.

Parameters:
  • goal_id (int) – Id of the goal for which the resource was posted
  • user_id (int) – Id of the user who posted the resource
  • title (str) – The title of the resource
  • link (str) – The resource’s web URL
  • topic (str) – The topic of the resource provided as text
  • description (str) – The description of the request
  • description – Estimate of the number of minutes required to assimilate the resource
Returns:

the id of the created message or None. None is returned if the resource was not found, or the specified goal does not exists in the database, or the specified user does not exist in the database
delete_resource(resource_id)¶

Delete the resource with id given as parameter.

Parameters:resource_id (int) – Id of the resource to remove.
Returns:True if the resource has been deleted, False otherwise
get_resource(resource_id)¶

Extracts a resource from the database.

Parameters:resource_id (int) – Id of the resource to be retrieved
Returns:A dictionary with the format provided in _create_message_object() or None if the resource with target id does not exist.
get_resources(goal_id, user_id, number_of_resource, max_length)¶

Return a list of all the resources in the database filtered by the conditions provided in the parameters.

Parameters:
  • goal_id (int) – Search resources of the goal with the given goal_id. If this parameter is None, it returns the resources of all the goals in the system.
  • user_id (int) – Search resources of the user with the given user_id. If this parameter is None, it returns the resources of all the users in the system.
  • number_of_resource (int) – Sets the maximum number of resources to return in the list. If set to None, there is no limit.
  • max_length (int) – All resources with a required time to complete greater than max_length are removed. If is set to None, this condition is not applied
Returns:

A list of resources. Each resource is a dictionary of items as created by _create_message_object()
modify_resource(resource_id, rating)¶

Modify the rating of the resource with id resource_id

Parameters:
  • resource_id (int) – The id of the resource to modify.
  • rating (float) – The resource’s rating
Returns:

The id of the modified resource or None. None is returned if the resource was not found or if the rating parameter is not a float value.
